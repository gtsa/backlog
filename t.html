Conflict Period Insight (RU–UA)

  • Interactive time-bar to select start/end dates of the conflict
  • Outputs territorial change totals (gains/losses) for both sides
  • Shows prewar / estimated population exposure for the chosen period
  • Optional map overlay for visual change
  • Target: web + mobile (responsive)

YouTube Transcript Fetcher

  • Option to output raw JSON, plain text, or SRT-style
  • Supports language detection / fallback
  • Can be used as a backend microservice
  • Potential: summarization later

API Concepts

TimelineAPI (For ChronoQuest)

  • /events?year=1905 - Get events from a specific year
  • /events?location=Europe - Get events by region
  • /events?range=1800-1900 - Get events within a timeframe

TaskFlow API (For Backlog App)

  • /tasks - Get all tasks
  • /tasks/{id} - Get details of a specific task
  • /tasks/create - Add a new task

NoteSync API (For Notes App)

  • /notes - Retrieve all notes
  • /notes?tag=work - Get notes by category
  • /notes/create - Add a new note

ScrobbleSync API (For Music Scrobbler)

  • /track/now-playing - Fetch current song
  • /track/history?user=123 - Get playback history
  • /track/stats?user=123 - Generate listening habits

WatchFinder API (For Movie List App)

  • /movies?title=Inception - Search for a movie
  • /movies/{id}/streaming - Find where to watch
  • /movies/trending - Get trending movies

ConflictData API (RU–UA)

  • /conflict/period?from=2022-02-24&to=2022-04-30 – Territorial deltas in period
  • /conflict/actors – List of actors/controlled areas
  • /conflict/population?date=2022-02-23 – Prewar population baseline
  • /conflict/map?from=...&to=... – GeoJSON for map overlay

YouTube Transcript API

  • /transcript?url={youtubeUrl} – Get transcript for a video
  • /transcript/{videoId}?lang=en – Force transcript language
  • /transcript/{videoId}/raw – Raw provider response
  • /transcript/{videoId}/segments – Time-coded segments
