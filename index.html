API Concepts

TimelineAPI (For ChronoQuest)

  • /events?year=1905 - Get events from a specific year
  • /events?location=Europe - Get events by region
  • /events?range=1800-1900 - Get events within a timeframe

TaskFlow API (For Backlog App)

  • /tasks - Get all tasks
  • /tasks/{id} - Get details of a specific task
  • /tasks/create - Add a new task

NoteSync API (For Notes App)

  • /notes - Retrieve all notes
  • /notes?tag=work - Get notes by category
  • /notes/create - Add a new note

ScrobbleSync API (For Music Scrobbler)

  • /track/now-playing - Fetch current song
  • /track/history?user=123 - Get playback history
  • /track/stats?user=123 - Generate listening habits

WatchFinder API (For Movie List App)

  • /movies?title=Inception - Search for a movie
  • /movies/{id}/streaming - Find where to watch
  • /movies/trending - Get trending movies
